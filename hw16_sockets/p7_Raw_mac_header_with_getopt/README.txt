Для корректной работы программы с getopt() необходимо запускать скомпилированную программу через sudo, передавая IP и MAC адреса сервера и клиента, а также в качестве последнего параметра — название сетевого интерфейса.
Пример:
sudo ./client -c 192.168.0.1 -s 192.168.0.2 -C 00:11:22:33:44:55 -S 66:77:88:99:aa:bb -i enp34s0
